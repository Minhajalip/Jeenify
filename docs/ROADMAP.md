# 📅 Project Roadmap
 
Jeenify – 7 Day Development Plan

This roadmap outlines the development schedule for building the Jeenify – Student Attendance & Assessment Management System.

The project will be completed over 7 days, with tasks distributed across frontend, backend, database, testing, and documentation teams.

---

# Day 1 – Project Setup & Planning

## Goals

Establish the project structure and development workflow.

### Tasks

- Create GitHub repository
- Setup repository branches
  - main
  - develop
  - frontend
  - backend
  - database
  - docs
  - testing
- Setup project folder structure
- Define database schema
- Setup development environments
- Assign tasks to teams
- Create documentation files

#### Deliverables:

- Project repository ready
- Initial documentation completed
- Development workflow established

---

# Day 2 – Authentication System

## Goals

Implement role-based login system.

### Backend Tasks

- Create users table
- Implement authentication logic
- Password hashing
- Session management

### Frontend Tasks

- Create login page
- Login form validation
- Error handling

#### Features Implemented

- Admin login
- Teacher login
- Student login
- Role-based redirection

##### Deliverables:

- Functional authentication system
