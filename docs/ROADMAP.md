# Project Roadmap
## Jeenify – 10 Day Development Plan

This roadmap outlines the planned development schedule for the Jeenify project.

## Day 1 – Project Setup
- Create GitHub repository
- Setup project folder structure
- Assign team roles
- Setup database design draft

## Day 2 – Database & Backend Setup
- Create MySQL schema
- Setup backend project structure
- Establish database connection

## Day 3 – Authentication Module
- Implement login page (frontend)
- Implement login logic (backend)
- Test authentication

## Day 4 – Student Management Module
- Add student form
- Implement student CRUD operations
- Connect frontend with backend

## Day 5 – Attendance Module
- Attendance marking interface
- Backend attendance logic
- Database integration

## Day 6 – Assessment Module
- Marks entry interface
- Backend logic for assessments
- Store marks in database

## Day 7 – Dashboard & Reports
- Create dashboard UI
- Implement attendance statistics
- Implement performance reports

## Day 8 – Integration
- Connect all modules
- Fix integration issues

## Day 9 – Testing
- Functional testing
- UI testing
- Bug fixing

## Day 10 – Finalization
- Documentation completion
- Final system testing
- Deployment & demo preparation
